VLSM-Based Corporate Network with Dual ISP Failover

VLSM-Based Corporate Network with Dual ISP Failover

Packet Tracer Project

This project involves designing and configuring a corporate network using Variable Length Subnet Masking (VLSM) to minimize IP address waste. It includes the implementation of redundant dual ISP connections to ensure continuous internet connectivity through automatic failover mechanisms.

Default routing was set up to handle traffic efficiently, ensuring reliable and seamless internet access even if one ISP connection fails. The network design emphasizes scalability, security, and fault tolerance.

Tools & Technologies used: Cisco Packet Tracer for simulation, VLSM subnetting principles, dual ISP failover configuration, routing protocols.
