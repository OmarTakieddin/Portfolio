Personal Portfolio Website

HTML | CSS

This project involved designing and developing a personal portfolio website using HTML and CSS to showcase my skills, projects, and professional background in a responsive and visually appealing format.

The website structure was hand-coded with semantic HTML elements for accessibility and maintainability, while modern CSS techniques were used for layout, styling, and responsiveness across devices. After completion, I shared the site with friends to gather feedback and also created custom portfolio sites for them based on their needs.

Tools & Technologies used: HTML5, CSS3, responsive design principles, GitHub for version control and hosting it on a free site.
